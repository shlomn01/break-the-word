# שבור את המילה — Break the Word

משחק קריאה רטרו בעברית: שובר את קיר הלבנים שמסתיר את המילה / המשפט / הפיסקה,
ובסוף כל שלב עונה על שאלת הבנה. 20 שלבים, 5 סצינות שמתחלפות (חלל, ים, יער, שקיעה, מדבר),
3 רמות קושי, ומוזיקת רקע.

**🕹 המשחק החי**: ראה את ה-URL בלשונית [Pages](#) של מאגר זה.

## איך לשחק

- **חיצים** — להזיז את המחבט
- **רווח** — לשגר את הכדור
- **M** — להשתיק / להחזיר את הקול

המטרה: שבור את כל הלבנים שמכסות את הטקסט עד שהוא מתגלה במלואו, וענה על שאלת ההבנה
שמופיעה בסוף השלב.

## טכני

- **Single-file static**: `index.html` בלבד עם תיקיית `assets/`
- **קנבס 768×600** עם רינדור פיקסל-ארט נקי
- **Web Audio** מסונתז לאפקטי המשחק (טניס, שבירה, קירות)
- **Heebo** ל-HUD (קריא בעברית); הטקסט הנסתר נחתך מאחורי קיר לבנים
- **Aseprite Lua** ייצר את הספרייטים (כדור, מחבט, מקשים) — הסקריפטים ב-`scripts/aseprite/`
- אין שלב build — פותח ישירות את `index.html` ומשחק

## ייחוסים

- מוזיקת הרקע נוצרה ב-Suno
